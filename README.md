# Smart-Helmet-Safety-System-with-Alcohol-Eye-Blink-Helmet-Detection
An Arduino-based smart helmet safety system that integrates alcohol detection, eye-blink monitoring, and helmet-wearing verification using MQ-5, IR, and ultrasonic sensors. The system triggers alarms and disables the motor when unsafe conditions are detected, ensuring enhanced rider safety and accident prevention.
